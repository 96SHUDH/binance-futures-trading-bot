# 📈 Binance Futures Trading Bot (Testnet)

## 🧾 Overview

This is a Python-based CLI trading bot that interacts with the Binance Futures Testnet (USDT-M).  
It allows users to place MARKET and LIMIT orders directly from the command line.

The project is designed with a clean modular structure, proper logging, and error handling to simulate a real-world backend system.

---

## 🚀 Features

- Place MARKET orders
- Place LIMIT orders
- Supports both BUY and SELL
- CLI-based input using argparse
- Input validation
- Logging of requests, responses, and errors
- Clean and modular code structure

---

## 🛠️ Tech Stack

- Python 3.x
- python-binance
- argparse
- logging
